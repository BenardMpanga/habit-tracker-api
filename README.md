# Habit Tracker API

A small FastAPI project for tracking habits. The app exposes a simple REST API that lets you list, create, read, update, partially update, and delete habit records.

Data is stored in a local SQLite database named `habits.db`, so records remain available after the server restarts.

## What Is In The Project

- `main.py` - the FastAPI application, database setup, models, and API routes.
- `requirements.txt` - Python dependencies for running and testing the app.
- `tests/` - API tests for the main CRUD behavior.
- `.gitignore` - ignores the local `venv/` virtual environment.
- `venv/` - local Python environment, not meant to be committed.

## Data Model

Each task has this shape:

```json
{
  "id": 1,
  "title": "Drink water",
  "status": "completed"
}
```

Fields:

- `id` - integer identifier generated by the API.
- `title` - habit name.
- `status` - habit state. Allowed values are `"pending"`, `"completed"`, and `"skipped"`. Defaults to `"pending"` if not provided.

## API Endpoints

| Method | Path | Description |
| --- | --- | --- |
| `GET` | `/tasks` | Get all tasks |
| `POST` | `/tasks` | Create a new task |
| `GET` | `/tasks/{task_id}` | Get one task by ID |
| `PUT` | `/tasks/{task_id}` | Replace an existing task |
| `PATCH` | `/tasks/{task_id}` | Partially update a task |
| `DELETE` | `/tasks/{task_id}` | Delete a task |

## Getting Started

Create and activate a virtual environment:

```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
```

Install the dependencies:

```powershell
pip install -r requirements.txt
```

Run the API:

```powershell
uvicorn main:app --reload
```

Then open the interactive API docs:

```text
http://127.0.0.1:8000/docs
```

## Example Requests

Create a new task:

```powershell
Invoke-RestMethod -Method Post `
  -Uri http://127.0.0.1:8000/tasks `
  -ContentType "application/json" `
  -Body '{"title":"Drink water","status":"pending"}'
```

Get all tasks:

```powershell
Invoke-RestMethod http://127.0.0.1:8000/tasks
```

Delete a task:

```powershell
Invoke-RestMethod -Method Delete http://127.0.0.1:8000/tasks/3
```

## Running Tests

```powershell
pytest
```
